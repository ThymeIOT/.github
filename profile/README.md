# Thyme

> _noun : a low-growing aromatic plant of the mint family. The small leaves are used as a culinary herb and the plant yields a medicinal oil._

The purpose of this organisation is to hold repositories for the Thyme organisation, developing the next generation of embedded IOT devices.

Features the following repositories:
- [Thyme Capsule](https://github.com/ThymeIOT/Capsule)
- [The Conservatory](https://github.com/ThymeIOT/Conservatory)
